## Our 1st Database:

1. `CREATE DATABASE db_name; `  
Create new database
Example:

```sql
CREATE DATABASE college;
```

2. `DROP DATABASE db_name;`  
Delete existing database
Example:

```sql
DROP DATABASE college;
```

3. `USE db_name;`  
Select the database to work with  
All table-related operations happen inside the selected database.
Example:

```sql
USE college;
```
