## Sigma Prime Note

Selects & Show data from the DB

Syntax:
```code
SELECT col1, col2 FROM table_name;
```
Exmaple:
```sql
SELECT id, name FROM user;          -- means show id, name column from 'user' table
```
---

Syntax: (to show all columns)
```code
SELECT * FROM table_name;
```
