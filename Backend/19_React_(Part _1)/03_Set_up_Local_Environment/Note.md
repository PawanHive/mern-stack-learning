# ⚛️ React Setup using Vite (Step-by-Step Guide)

---

## 📌 What is Vite?
Vite is a **fast build tool** for modern frontend development.  
It makes React projects start and reload **very quickly** compared to older setups like CRA.

---

## 🧰 Prerequisites

Make sure you have installed:

- `Node.js` (LTS version)
- `npm` (comes with Node.js)

Check versions:
```bash
node -v
npm -v
```

---

## 🚀 Step 1: Create React Project using Vite

Run this command:

```bash
npm create vite@latest
```

---

## 🧾 Step 2: Project Setup Questions

You will be asked:

- Project name → (your choice)
- Framework → **React**
- Variant → **JavaScript** (or TypeScript if you want)

Example:
```
✔ Project name: my-app
✔ Framework: React
✔ Variant: JavaScript
```

---

## 📂 Step 3: Move into Project Folder

```bash
cd my-app
```

---

## 📦 Step 4: Install Dependencies

```bash
npm install
```

---

## ▶️ Step 5: Run Development Server

```bash
npm run dev
```

---

## 🌐 Step 6: Open in Browser

After running, you will see something like:

```
http://localhost:5173/
```

Open it in your browser.

## 🔥 Useful Commands

Start server:
```bash
npm run dev
```

Build project:
```bash
npm run build
```

Preview build:
```bash
npm run preview
```
